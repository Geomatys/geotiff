= Detailed Test Suite (Normative)

== General Approach

This GeoTIFF standard specifies an encoding of information for exchange between software components.
As such, this Detailed Test Suite can only validate the conformance of a specific GeoTIFF document.
In most cases, GeoTIFF conformance will be incorporated into the conformance testing of a software component.
These components fall into two categories; GeoTIFF readers and GeoTIFF writers.

Conformance of GeoTIFF readers will validated by the successful processing of a representative sample of validated GeoTIFF files.

Conformance of GeoTIFF writers will be validated by the successful validation of a representative sample of GeoTIFF files generated by that software component.
